Group Level

Description

The Group Level page allows you to configure a group section logged in the TDMS file. Groups are a collection of channels in a TDMS file and provide a way of organising your data channels.

Configuration

Available for the user to configure is:
- The meta data associated on the group level in the TDMS file as a key value pairing (property name, property value).
- The name of the group as it is recorded in the TDMS (always matches as it appears in the Custom Device).
