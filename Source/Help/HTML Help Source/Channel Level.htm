Channel Level

Description

The Channel Level page allows you to configure a channel logged in the TDMS file. These are channels you want recorded at the rate of the controller and single point values.

Configuration

Available for the user to configure is:
- The meta data associated on the channel level in the TDMS file as a key value pairing (property name, property value).
- The Decimation of the specific channel, i.e. downsampling from the controller rate.
- The name of the channel as it is recorded in the TDMS (always matches as it appears in the Custom Device).

You set the source of data by mapping a channel to this channel in the System Definition mappings.
