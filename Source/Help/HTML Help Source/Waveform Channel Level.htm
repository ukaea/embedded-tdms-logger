Waveform Channel Level

Description

The Waveform Channel Level page allows you to configure a waveform channel logged in the TDMS file. These are channels you want recorded that exist as waveforms in the target controller and record at varying rates to the controller.

Configuration

Available for the user to configure is:
- The waveform being recorded
- The meta data associated on the channel level in the TDMS file as a key value pairing (property name, property value).
- The Decimation of the specific channel, i.e. downsampling from the waveform rate.
- The name of the channel as it is recorded in the TDMS (always matches as it appears in the Custom Device but does not necessarily match to the actual waveform source).

You set the source of waveform data within the channel page itself rather than mappings, unlike the typical channels in this custom device.
