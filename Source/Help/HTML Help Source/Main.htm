File/Custom Device Level Configuration

Description

The Main page allows you to configure file level properties of the TDMS logs as well as the behaviour of the custom device during execution.br>

Configuration

Available for the user to configure is:
- The name of the custom device (not the same as the file name of the TDMS logs).
- The filename as derived for the TDMS files, this can be a expression which incorporates the time and date into the file name construction during logging.
- The directory for which to log files into.
- The meta data associated on the file level in the TDMS file as a key value pairing (property name, property value).
- Conditions for which to segment the TDMS file into seperate files during logging, namely by size in MB or by time in seconds. A common use case is to segment the file to every hour.
